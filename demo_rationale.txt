RoPE Playground: Rotation Is All You Need is designed to teach one focused idea: RoPE encodes token position by rotating query/key vectors, so attention can depend on relative position.

I organized the demo as short interactive steps instead of a long explanation. Step 1 motivates why order matters with two sentences that use the same words but mean different things. Step 2 shows the core 2D rotation with a draggable position slider and the rotation formula. Step 3 demonstrates the main aha moment: shifting both query and key keeps their relative offset, so the attention score pattern stays tied to relative distance. Step 4 explains the high-dimensional version with four small clocks, making 2D pairs and theta values visual instead of abstract. Step 5 connects this to attention scores with clickable query tokens and a long periodic example. Step 6 adds a real locally generated tiny training replay for an A-before-B task, comparing no position, sinusoidal, learned absolute, and RoPE.

The interface is intentionally concise: each step has one main sentence, one interaction, and one visual target. I used consistent colors for query, key, position, RoPE, and comparison methods. I also added fullscreen mode so the demo can be shown in class without scrolling.

The training panel is labeled as a tiny experiment, not a benchmark. Its data is generated by a local TensorFlow script and saved as JSON/JS replay data, so the browser demo remains fast and deterministic while still reflecting an actual small training run.
